# BRZ Tune Backups
These are tunes for first-gen Subaru BRZs (can also apply to the Toyota 86 or Scion FR-S).

## Tunes

### read.bin
This is an unmodified backup of the factory ECU calibration.

### Stage 1
This is based on the OFT Stage 1 tune, and is intended for a 91 octane+, stock airbox, header, and exhaust.

### Stage 2 Plus
This is based on the OFT Stage 2+ tune, and is intended for a 91 octane+, stock airbox, catless header, and any exhaust.
